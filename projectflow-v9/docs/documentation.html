03 · Dashboard

Visão Global — Todos os Projetos

O Dashboard V9 exibe todas as tarefas de todos os projetos simultaneamente, com 7 tipos de filtros e exportação PDF.

Filtros disponíveis

Por Projeto
Filtra por projeto ou exibe todos

Por Prioridade

Por Responsável
Membro da equipe
Por Cliente
Derivado do projeto vinculado

Intervalo de data de entrega

Exportação PDF

Botão "Exportar PDF" abre janela de impressão com todos os dados filtrados, incluindo KPIs, gráficos de barras e tabela completa de tarefas. Suporta impressão direta ou salvar como PDF pelo browser.

07 · Instalação

Como instalar

Estrutura de arquivos V9

projectflow-v9/
├── index.html                  ← Entrada principal
├── css/                        ← Estilos (inalterados)
├── js/
│   ├── board.js                ← Kanban base
│   ├── knowledge-base.js       ← Wiki editor
│   ├── ai-doc-engine.js        ← Doc IA
│   ├── pf-producao.js          ← Patches V8
│   ├── diagram-engine-v9.js    ★ NOVO: Fabric.js editor
│   └── pf-v9-core.js           ★ NOVO: Toast, Dashboard, Kanban V9
├── sql/
│   ├── producao.sql            ← Schema base (rodar primeiro)
│   └── v9-migration.sql        ★ NOVO: Migrações V9
└── docs/
    └── documentation.html      ← Este arquivo

Passos

  1. Criar projeto no supabase.com
  2. Executar producao.sql → depois v9-migration.sql no SQL Editor
  3. Servir a pasta localmente (Live Server, nginx, etc.)
  4. Clicar ⚙️ na tela de login → inserir URL e Anon Key do Supabase
  5. Ou usar o botão "Demonstração" para testar sem banco
🚀

CDN: Fabric.js 5.3.1 carrega via cdnjs.cloudflare.com automaticamente. Zero dependências npm, zero build step — 100% vanilla JS.
